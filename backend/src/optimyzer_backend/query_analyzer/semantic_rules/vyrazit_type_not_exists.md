# Тип в ВЫРАЗИТЬ() не существует в конфигурации

## Что не так

В конструкции `ВЫРАЗИТЬ(... КАК {{object_full_name}})` указан тип, которого нет в подключённой выгрузке.

## Почему это плохо

`ВЫРАЗИТЬ(Поле КАК Справочник.Х)` приводит составной тип к конкретному. Если тип не существует — запрос **не выполнится**.

Чаще всего такая ошибка встречается, когда:

1. **Запрос из другой конфигурации**, где этот тип был.
2. **Опечатка в имени** ссылочного типа.
3. **Тип удалён** из конфигурации, а запрос остался старым.

## Как исправить

1. Замените тип на корректный из списка выше.
2. Если приведение типа на самом деле не нужно — уберите `ВЫРАЗИТЬ` совсем.
3. Если у поля несколько возможных типов — приводите к каждому через отдельную ветку `ВЫБОР КОГДА ... ТОГДА ВЫРАЗИТЬ(... КАК ...)`.
