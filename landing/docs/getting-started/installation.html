Установка Optimyzer

Standalone-приложение, без серверов и баз 1С на стороне. Скачиваете установщик — двойной клик — готово.

Системные требования

  • Windows: 10 / 11 (x64), 8 ГБ RAM минимум, 16+ ГБ рекомендуется
  • macOS: 12 Monterey+, Apple Silicon или Intel x64
  • Linux: Ubuntu 22.04+ / Fedora 39+ / Arch, glibc 2.35+
  • Диск: 500 МБ под установку, плюс место под анализируемые архивы

Скачать

Все релизы — на странице GitHub Releases . Или через прямой редирект: optimyzer.pro/download.

Windows

  1. Скачайте Optimyzer_Setup_x.y.z_x64.msi
  2. Запустите установщик — Windows может предупредить о неизвестном издателе (исправим подписанным сертификатом в следующих релизах)
  3. После установки — иконка в меню «Пуск»

macOS

  1. Скачайте Optimyzer_x.y.z_universal.dmg
  2. Откройте DMG, перетащите Optimyzer в Applications
  3. При первом запуске: System Settings → Privacy & Security → Open Anyway (приложение пока не подписано Apple Developer ID)

Linux

Распространяется как AppImage:

chmod +x Optimyzer-x.y.z-x86_64.AppImage
./Optimyzer-x.y.z-x86_64.AppImage

Для интеграции в меню — используйте AppImageLauncher .

Первый запуск

При первом старте Optimyzer покажет welcome-окно и подскажет загрузить архив. Подробнее — Первый архив.
