Privacy & Data

Что хранится локально, что отправляется в облако, что попадает в Claude — без юридического тумана.

Локально хранятся

  • Загруженные архивы ТЖ — в виде DuckDB-индексов в %APPDATA%\Optimyzer\data (Windows) или ~/.optimyzer/data (macOS/Linux)
  • Кеш AI-ответов — в локальной SQLite
  • JWT токен лицензии — в localStorage (на v2 переедет в OS keychain)
  • Telemetry буфер до отправки — в localStorage

На наш сервер (api.optimyzer.pro) уходит

  • Email и Yandex ID (из OAuth) — для аккаунта
  • Hardware fingerprint (sha256 от platform+OS+random-uuid) — для идентификации устройства
  • Метаданные платежей и подписки
  • Запись о факте AI-операции: тип, timestamp, success — но НЕ содержимое
  • Анонимная telemetry — см. ниже

В Claude API уходит (только по нажатию «Объяснить через AI»)

  • Агрегированный контекст конкретной проблемы (тип события, суммарные метрики, топ-N нормализованных SQL, lock graph)

В Claude НЕ уходит

  • Содержимое значений параметров запросов
  • Имена баз 1С, имена пользователей 1С
  • Содержимое контекстов выполнения (Context поля целиком)
  • Сообщения исключений с конкретными данными
  • Любые данные из других сессий или других архивов

Анонимная telemetry — что собирается

Отправляются обезличенные метаданные использования продукта — без персональных данных, без содержимого ваших архивов. Помогают понять какие функции реально востребованы и куда направлять развитие.

Что собирается:

  • Размер загружаемых архивов в МБ
  • Количество событий после парсинга, время парсинга
  • Какие экраны посещаются, сколько раз нажимали AI
  • ОС, версия приложения, размер экрана
  • Конверсионные события: видел ли paywall, нажал ли upgrade

Что НЕ собирается:

  • Содержимое SQL-запросов
  • Имена баз, пользователей
  • AI-ответы
  • IP-адрес (только страна/регион в логах nginx, и то через 7 дней удаляется)

Telemetry удаляется автоматически через 90 дней. Связь с аккаунтом только для Pro юзеров — Free юзеры идут с anonymous device_fingerprint.

Удаление данных

Запрос на полное удаление аккаунта и связанных данных — пишите на hello@optimyzer.pro. Обработаем в течение 14 дней.

Где хранятся данные

VDS в РФ (Selectel / Beget / RuVDS — на момент написания, может измениться). Backup'ы — там же. Claude API — Anthropic в США (для AI-консультаций; согласие на это передаётся при первом вызове AI кнопки).
